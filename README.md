# Hydrological Spatial-Temporal Analysis of the Wesel-Datteln Canal System

## Overview

This project analyzes water level measurements from the Wesel-Datteln Canal network in Germany using R. The objective was to evaluate water level variation across multiple lock stations, compare upstream and downstream conditions, and visualize hydrological patterns through statistical and graphical analysis.

---

## Data Overview

### Data Source

Federal Waterways and Shipping Administration (WSV) via the Pegelonline Hydrological Information System.

### Study Period

January 1, 2026 – June 1, 2026

### Monitoring Stations

* Ahsen Schleuse OW (Upper Lock)
* Ahsen Schleuse UW (Lower Lock)
* Datteln Schleuse OW (Upper Lock)
* Datteln Schleuse UW (Lower Lock)
* Dorsten Schleuse OW (Upper Lock)

### Tools

* R
* RStudio
* Base R Graphics
* R Markdown

---

## Methodology

### 1. Data Import and Preparation

* Imported raw hydrological data using the German semicolon-separated format.
* Converted water level measurements to numeric values.
* Removed incomplete records and missing values.
* Prepared a clean dataset for analysis and visualization.

### 2. Statistical Analysis

* Generated summary statistics for all monitoring stations.
* Compared water level distributions across upstream and downstream lock locations.
* Examined variability, median levels, and distribution patterns.

### 3. Data Visualization

* Created station-level boxplots to compare water level distributions.
* Generated density plots to examine overall hydrological behavior.
* Exported visual outputs and analysis results through an R Markdown report.
